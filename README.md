<h1 align="center">⭐ Regedit Compilations ⭐</h1>

I'm presenting to you the registry compiled files that I created for use on my personal computer.

The content of regedit commands is set to Turkish. You can translate it to any language you want and use it.

You can use it on Windows 7/8/10/11 operating systems.

## 📌 Main Features
- Windows Explorer and right-click on the desktop to select `New Text Document` (edited)
- `Advanced User Accounts` have been added to the Control Panel
- Right-click on Desktop & Folder and Drive and add `Open Command Prompt Here` and `Open Command Prompt as Administrator Here` options.

## 📌 How to Use
You can use it by downloading the compilations.reg file to your computer or by downloading the file inside the releases folder and running it as administrator.
